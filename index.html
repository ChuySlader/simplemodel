Welcome to SimpleModel Page!

SimpleModel allows you to create and manage conections between the html data on your page and your server, SimpleModel create instances related to a predefined model which contains all the info and events related with the model, and let you manage the info and fields related with the instance.

You can create a new model as easy as:

var NewModel = new SimpleModel();

But this wont make anything :( so lets create a usefull example:

var MyUserModel = new SimpleModel({
    'url': '/YourUrlThatManageThisModel',
    'type': 'IdentifierForThisModel',
    'callbacks': {
        'afterPublish': function (data, self) {
            alert("Hey! You just published the instance: "+self.id)
        }
    },
    'handlers': {
        'errorHandler': function (error, data) {
            switch (error) {
            case 'ERROR_ON_PUBLISH_RESPONSE':
                (function (error) {
                    alert("Woops something went wrong on the server: " + error);
                })(data);
            }
        }
    },
    'fields': {
        'id': {
            'default': ''
        },
        'username': {
            'required': true
        }
    }
});

And now lets create a new instance of the prev model:

var info = jQuery("#myInfoContainer").find(".model-data");
var User = MyUserModel.create(info);

So now every time you modify a html input the model will publish automatically the data to your server under "/YourUrlThatManageThisModel/publish".
